# Credit-Score-Classification-With-Machine-Learning

\# Credit Score Classification



This project classifies users into different credit score categories

using machine learning based on financial attributes.

Publicly available financial dataset used for academic purposes.



\## Algorithm Used

\- Logistic Regression / Random Forest



\## Steps

1\. Data preprocessing

2\. Model training

3\. Evaluation
